**Ottawa EV Report DAO (Java)**

A Java application that uses the DAO design pattern, business layer,
and transfer objects to retrieve and display Ottawa EV and postal code data from a database.

**Features**

DAO pattern with interfaces and implementations
Business layer for report generation
Transfer objects for structured data handling
Database-driven reporting and formatted output

**Technologies**

Java
DAO Design Pattern
SQL / MySQL
Object-Oriented Programming (OOP)

**Note**

A configuration file for database access is required. Sensitive database 
credentials are not included in the repository.
