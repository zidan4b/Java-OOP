**Course Registration Servlet Lab (Java Web)**

A Java web application that allows users to register courses, store registration data in a database, 
and view submitted registrations through a browser.

**Features**

Course registration form with HTML and CSS
Java Servlets for request handling
DTO class for registration data
Database connectivity for storing and viewing records

**Technologies**

Java
Java Servlets
HTML / CSS
SQL / MySQL
DTO Pattern

**Note**

Database connection settings must be configured before running the application. Sensitive credentials are not included in the repository.
