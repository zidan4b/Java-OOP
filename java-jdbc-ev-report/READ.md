**Java JDBC EV Report (Java)**

A Java application that connects to a MySQL database, runs SQL join queries, retrieves metadata,
displays EV data by city and FSA, and calculates totals.

**Features**

Database connectivity using JDBC
SQL join queries and result processing
ResultSet metadata display
Formatted console output and totals calculation

**Technologies**

Java
JDBC
SQL / MySQL
Object-Oriented Programming (OOP)

**Note**

A database.properties file is required for database connection settings.
Real database credentials are not included in the repository.
